# Enhance Note Command

Este comando es un alias directo o envoltorio simplificado para ejecutar [Enhance By AI Command](./EnhanceByAiCommand.md) sobre la nota activa.

Su función principal es asegurar que la acción se realiza sobre un archivo Markdown válido antes de invocar la lógica de IA.

Ver [Enhance By AI Command](./EnhanceByAiCommand.md) para más detalles.
