# Obsidian commands

## Generate Missing Notes

Create notes for unresolved links in your current document.

## Enhance with AI

Improve or complete the body and metadata of your note using AI.

## Apply Template

Apply a template to your note, with AI-powered content generation if the template contains a `!!prompt`.

## Apply Template with URL

Apply a template using content from a website as additional context for the AI.

## Enrich with URL

Update your current note by fetching and analyzing content from a specified URL.

## Add Images

Quickly add images from your clipboard or a specific local folder to your note.

## Create Note from Images

Create a new note automatically by analyzing or transcribing images (from a folder or clipboard).

## Apply Template from Images

Use images as the primary source of truth to fill out a structured template using AI.

## Relocate

Automatically move the current note to its correct folder based on designated frontmatter fields.

## Create Reciprocal Links

Manage two-way relationships between notes (e.g., automatically linking back from a person's note to the meeting where they were mentioned).

## Analyze and Link Entities

Let AI scan your text to identify and link important entities like people, organizations, or concepts.

## Generate Missing Notes from List Field

Mass-create notes for all items listed in a specific frontmatter metadata field.

## Insert Link to Selected Photo

Grab a link to the photo currently selected in the macOS Photos app and insert it into your note.

## Open Linked Photo

Instantly open the photo referenced in your note's metadata in the macOS Photos app.

## Natural Language Search

Find notes in your vault using conversational queries instead of complex search syntax.

## Tokenize and Create Notes

Extract key terms or vocabulary from your text and create dedicated notes for them in a dictionary.

## Hide/Show Empty Properties

Toggle the visibility of empty metadata fields in the Obsidian editor to keep your view clean.
